# 化学品物性数据Excel生成器

从任意格式的化学品输入出发，生成一份**全程可追溯、带来源注释**的Excel物性数据表。
最终产物包含四个Sheet：物性数据主表 + 介质-设备对照表（可选） + 数据来源汇总表 + 化学品清单（可选）。

---

## 0. 输入识别

用户输入可能是以下任意形态，**不要求用户预先整理**：

| 形态 | 特征 | 处理策略 |
|------|------|----------|
| **脏数据** | 大段工艺描述、SDS全文、会议记录、混杂文字 | Step 1: 提取 |
| **干净列表** | 已整理的化学品名称列表 | 跳过 Step 1 |
| **半成品** | 名称+部分属性已知，缺其他属性 | Step 1 跳过已有属性，仅补缺 |
| **设备一览表** | 包含"物料介质"/"介质"列的设备清单xlsx | Step 0.5: 从设备表提取 |

收到输入后，先判断属于哪种形态，再决定从哪一步开始。

---

## 0.5 设备一览表介质提取（当输入为设备表时）

当用户提供设备一览表（含设备位号、名称、工段、物料介质等列）时：

### 介质列识别
设备一览表通常按设备类型分sheet（如反应釜、塔、换热器、容器、泵等），每个sheet的列结构不同。**不要预设列位置**，应按以下方法动态识别：

1. 读取每个sheet的第2~3行表头行（含合并单元格）
2. 按关键字匹配介质列：搜索包含 `介质`、`物料`、`主要介质`、`主要成分` 的表头单元格
3. 同时识别位号列（含 `位号`、`设备位号`）、名称列（含 `设备名称`、`名称`）、工段列（含 `工段`、`工序`）
4. 换热器类设备通常有管程/壳程两组介质，均需提取

### 化学品名称归并

设备表中的介质写法常有**变体、错别字、浓度前缀**，需做规范化映射：

**常见归并模式：**
- 异体字：噁唑酮 = 嗯唑酮 = 唑酮；吡咯腈 = 吡咯晴
- 同义变体：丙烯腈 = 丙烯晴 = 2-氯丙烯腈 = 2-氯丙烯睛
- 浓度前缀剥离：27.5%双氧水 → 过氧化氢(双氧水)；32%液碱 → 氢氧化钠
- 少量/微量前缀剥离：少量甲苯 → 甲苯；微量氯化氢 → 氯化氢(气)
- 别名归一：氯甲醚 → 氯甲基乙醚；二氯氧基甲烷 → 二乙氧基甲烷

**公用工程/传热介质不计入工艺化学品：**
水、循环水、7℃冷水、-10℃冷冻水、蒸汽、80℃热水、导热油、乙二醇冷冻液、矿物油、白油、少量水、杂质、酸性水等。

### 工程名称标准化
为每个化学品指定一个**唯一的工程名称**（canonical name），用于后续所有sheet。

## 1. 化学品提取（仅脏数据需要）

从文本中识别所有化学品，包括：
- 原料、试剂、溶剂、催化剂
- 反应中间体、副产物
- 产品、目标化合物
- 辅助介质（导热油、冷媒等，标注"因牌号而异"）

**提取规则：**
- 同一物质的不同写法合并（如"盐酸"/"氯化氢水溶液"→按工艺角色分别列出）
- 气体、液体、固体均纳入
- 水、蒸汽、空气等极普通物质标注后列出，但可与用户确认是否纳入
- 提取完成后列出清单，**请用户确认或增删**后再进入下一步

**输出格式（内部）：**
```
序号 | 化学品名称 | 分类 | 涉及工段 | 出现设备数(去重) | 原表介质写法(变体归并)
```

## 2. 物性数据采集

对每种化学品，通过 WebSearch + WebFetch 查询以下14个字段：

| 字段 | 英文 | 单位 | 首选来源 |
|------|------|------|----------|
| CAS号 | CAS Number | — | PubChem |
| 熔点 | Melting Point | °C | NIST WebBook |
| 沸点 | Boiling Point | °C | NIST WebBook / ChemicalBook |
| 闪点 | Flash Point | °C | Sigma-Aldrich SDS |
| 密度 | Density | g/cm³ 或 g/L | PubChem |
| 常温物态 | State at RT | 固/液/气 | ChemicalBook / SDS |
| 火灾危险性类别 | Fire Hazard Class | 甲/乙/丙/非可燃 | GB 50016-2014 |
| 水溶性 | Water Solubility | 描述 | PubChem |
| 密度>水？ | Density > Water | 是/否 | 计算 |
| 禁用水灭火？ | Water Extinguishing Prohibited | 是/否 | CAMEO / SDS |
| 爆炸极限 | Explosion Limits | LEL%~UEL% | NIOSH Pocket Guide |
| 蒸汽>空气？ | Vapor > Air | 是/否(MR) | NIST WebBook |
| 职业接触限值 | OEL | mg/m³ (TWA/STEL/MAC) | GBZ 2.1-2019 |
| 危化品类别 | Danger Category | 易燃液体/腐蚀品等 | 危险化学品目录(2015版) |

**数据采集原则：**
- 每个字段从不同来源获取，确保同一行不同列来源不同
- 中国法规数值优先（火灾危险性、OEL、危化品类别）
- 不可燃物质的闪点/爆炸极限标注"—"
- 未知或因牌号而异的标注"因牌号而异"或"数据未公开"
- **工艺专有中间体（无公开CAS号）：物理常数标"数据未公开"，火灾类别标"丙类(估)"或"待定"，来源注明"工艺专有中间体，无公开数据，按有机物估计"——绝不编造数值**

### 并行Agent批次查询策略

将化学品分为4~6批，每批用独立Agent并行查询。**Agent必须将结果写入JSON文件**（而非直接返回文本，避免上下文截断）：

```
批次1(溶剂/通用): Agent写入 → _props_b1.json
批次2(酸/试剂):   Agent写入 → _props_b2.json
批次3(胺/气体):   Agent写入 → _props_b3.json
批次4(中间体):    Agent写入 → _props_b4.json
```

**Agent Prompt关键指令：**
- 明确列出每种化学品的中文名（JSON key），要求Agent原样使用
- 要求Agent用Write工具写入指定路径的UTF-8 JSON文件
- 每种化学品15个字段（CAS~危化品类别 + 来源），全部字符串值
- 用"—"标记不适用项，"数据未公开"标记查不到的数据
- Agent完成后返回"DONE bN"确认

**合并后验证：** 检查每个JSON文件是否可解析、条目数是否正确、是否覆盖所有化学品。

## 2.5 数据源完整URL与查询指南

### 国际数据库

| 来源 | 完整URL | 查询方法 |
|------|---------|----------|
| **PubChem** | `https://pubchem.ncbi.nlm.nih.gov/compound/<CAS号>` | 用CAS号搜；页面Sections: Chemical & Physical Properties, Solubility, Safety & Hazards |
| **NIST Chemistry WebBook** | `https://webbook.nist.gov/cgi/cbook.cgi?ID=<CAS号>&Units=SI` | 直接用CAS号查，页面包含：Phase change data (熔沸点), Gas phase thermochemistry (蒸气密度) |
| **ChemSpider** | `http://www.chemspider.com/Search.aspx?q=<CAS号>` | 搜索CAS号，Properties部分含物态、密度 |
| **ChemicalBook** | `https://www.chemicalbook.com/ProductChemicalProperties_CB<编号>_EN.htm` | 搜索CAS号或中文名，首页即显示熔点/沸点/闪点/密度/物态表 |
| **Sigma-Aldrich SDS** | `https://www.sigmaaldrich.com/` → 搜索产品 → Download SDS | SDS中Section 9: Physical & Chemical Properties (闪点/密度/物态) |
| **CAMEO Chemicals** | `https://cameochemicals.noaa.gov/chemical/<CAS号>` | Reactivity: 遇水反应性；Flammability: 爆炸极限；Health: 禁用水？ |
| **NIOSH Pocket Guide** | `https://www.cdc.gov/niosh/npg/npgd<编号>.html` | Explosive limits (LEL/UEL)；IDLH值；Relative Density (蒸气密度MR) |
| **FAO/WHO JMPR** | `https://www.fao.org/agriculture/crops/` | 农药原药物化及毒理数据 |

### 中国法规标准

| 来源 | 查询网址 | 查询方法 |
|------|----------|----------|
| **GB 50016-2014** 《建筑设计防火规范》 | `https://openstd.samr.gov.cn/` | **表3.1.1**：火灾危险性分类——按闪点划分甲乙丙类 |
| **GBZ 2.1-2019** 《工作场所有害因素职业接触限值》 | `https://openstd.samr.gov.cn/` | **表1**：OEL值——TWA/STEL/MAC |
| **GB 15603-1995** 《常用危险化学品贮存通则》 | `https://openstd.samr.gov.cn/` | 附录：遇湿易燃物质、禁止水灭火物质清单 |
| **GB/T 42594-2023** 《承压设备介质危害分类导则》 | `https://openstd.samr.gov.cn/` | 附录A：毒性分级——I级/II级气体蒸气 |
| **GB/T 50493-2019** 《石油化工可燃气体和有毒气体检测报警设计标准》 | `https://openstd.samr.gov.cn/` | **附录B**：常见有毒气体、蒸气特性表 |
| **HG/T 20660-2017** 《压力容器中化学介质毒性危害和爆炸危险程度分类标准》 | `https://openstd.samr.gov.cn/` | **附录A**：毒物危害指数I/II级物质（气体/蒸气） |
| **《高毒物品目录》** (卫法监发[2003]142号) | `https://www.mem.gov.cn/` | 54种高毒物质MAC/TWA/STEL |
| **《危险化学品目录》(2015版)** | `https://www.mem.gov.cn/` | 是否列入；备注栏标注"剧毒" |

### 查询优先级

```
1. 中国法规有明确数值 → 优先采用（OEL、火灾类别、危化品类别）
2. 国际数据库有数据 → 采用（物化属性：熔点、沸点、密度等）
3. 供应商SDS有数据 → 参考（闪点、遇水反应性等）
4. 以上均无 → 标注"数据未公开"或"因牌号而异"
```

### 查询关键词模板

WebSearch 查询时使用以下关键词模式：
```
# 物化属性
"<中文名> CAS <CAS号> 熔点 沸点 闪点 密度"
"<英文名> boiling point melting point flash point density SDS"

# 安全数据
"<中文名> 安全技术说明书 SDS 爆炸极限"
"<英文名> NIOSH pocket guide explosion limits"

# 中国法规
"GBZ 2.1-2019 <化学名> 职业接触限值"
"危险化学品目录 <化学名> CAS <CAS号>"
```

## 3. 法规分类判定

### 3.1 火灾危险性（GB 50016-2014 表3.1.1）

| 分类 | 判定依据 | 颜色 |
|------|----------|------|
| **甲类** | 闪点 < 28°C 的液体；爆炸下限 < 10% 的气体 | 红色 FFCCCC |
| **乙类** | 28°C ≤ 闪点 < 60°C 的液体 | 橙色 FFE0B2 |
| **丙类** | 闪点 ≥ 60°C 的液体；可燃固体 | 黄色 FFF9C4 |
| **非可燃** | 不可燃液体/固体/气体（闪点不适用） | 白色 FFFFFF |

**仅甲/乙/丙着色，其余统一标"非可燃"白色底。**

**特殊物质注意：**
- 遇水反应品（如PCl₃）：不可燃，不归入甲乙丙，标"非可燃"
- 氧化剂（如H₂O₂、Br₂）：不可燃，标"非可燃"
- 强酸（如H₂SO₄、HCl）：不可燃，标"非可燃"

### 3.2 职业接触限值（GBZ 2.1-2019）

**优先读取本地参考文件：** `gbz2.1-oel-reference.md`（位于本skill同目录），包含约300种化学品的CAS号+MAC/TWA/STEL。按CAS号比对即可，无需联网。

> 注意：本地参考数据基于GBZ 2.1-2007版提取，2019版为现行版本。若化学品未在本地文件中找到，应联网查询GBZ 2.1-2019确认。

- MAC（最高容许浓度）：急性毒性物质
- TWA（时间加权平均）：8h工作日平均
- STEL（短时间接触）：15min短时间
- 单位统一为 mg/m³

### 3.3 危险化学品目录（2015版）

**优先读取本地参考文件：** `hazcat-2015-reference.md`（位于本skill同目录），包含2674种化学品的CAS号+危险性类别+备注（含"剧毒"标记）。按CAS号比对即可，无需联网。

**填写规则（严格遵守）：**
- **在目录中：** 必须原样誊写目录中的"危险性类别"文本（如"易燃液体,类别2"、"急性毒性-经口,类别3\*"、"皮肤腐蚀/刺激,类别1A"等），不得简化或改写。同一CAS在目录中有多行条目时，合并所有条目的危险性类别，用"；"分隔。
- **不在目录中：** 填写 `未录入《危险化学品目录》`，后面可附加联网查询到的危险性分类作为补充参考。
- **禁止使用笼统描述：** 不得写"易燃液体"、"腐蚀品"、"有毒品"等笼统词，必须用目录原文的GHS分类格式（含类别编号）。

### 3.4 有毒气体检测目录

对照以下四个来源，判定每种化学品是否列入有毒气体检测目录。

**优先读取本地参考文件：** `toxic-gas-directory-reference.md`（位于本skill同目录），包含完整的高毒物品目录54种、GB/T 50493-2019附录B 3种、HG/T 20660-2017附录A 66种、剧毒物质148种CAS号。直接按CAS号比对，无需联网。

| 来源 | 性质 | 文案 |
|------|------|------|
| 《高毒物品目录》(卫法监发[2003]142号) | 54种高毒物质 | **是，应设报警，来自《高毒物品目录》** |
| GB/T 50493-2019 附录B | 常见有毒气体蒸气特性表 | **是，暂不设报警，来自GB/T 50493-2019** |
| HG/T 20660-2017 附录A | 毒物危害指数I/II级（剔除剧毒后66种，标黄） | **是，暂不设报警，来自HG/T 20660-2017** |
| 《危险化学品目录》(2015版)备注"剧毒" | 剧毒物质 | 与高毒物品目录合并表述 |

**文案规则（严格遵守）：**
- 命中高毒物品目录或危化品目录剧毒：`是，应设报警，来自《高毒物品目录》`
- 命中GB/T 50493-2019附录B或HG/T 20660-2017附录A（标黄物质）：`是，暂不设报警，来自GB/T 50493-2019` 或 `是，暂不设报警，来自HG/T 20660-2017`
- 未命中任何来源：`否`

**着色：** "暂不设报警"的行用黄底（FFF9C4），其余不着色。

**来源注释：** 每个单元格批注中写明具体出处，如"HG/T 20660-2017附录A"或"《高毒物品目录》(卫法监发[2003]142号)"。

## 4. Excel 生成

使用 Python + openpyxl 生成，**必须包含以下内容**：

### Sheet 1：物性数据主表

**列结构（17列，含有毒气体列）：**
```
序号 | 化学品名称 | CAS号 | 熔点(°C) | 沸点(°C) | 闪点(°C) | 密度 | 常温物态 | 火灾危险性类别 | 水溶性 | 密度>水？ | 禁用水灭火？ | 爆炸极限(LEL~UEL) | 蒸汽>空气？ | 职业接触限值(mg/m³) | 危化品类别 | 是否被列入有毒气体检测目录
```

**格式要求：**
- 表头：深蓝底白字，微软雅黑 10pt 加粗
- 数据行：微软雅黑 9pt，自动换行
- 统一行高：30px
- 列宽紧凑但不遮盖：5~26字符
- 冻结首行

**颜色标注（仅3列着色）：**
- 化学品名称列、CAS号列、火灾危险性类别列
- 甲类=红底 / 乙类=橙底 / 丙类=黄底 / 非可燃=白底

**来源注释（关键）：**
- 每个数据单元格添加 Excel Comment
- 同一行不同列的注释来源**必须不同**
- 格式：`数据来源:\n<机构名> (<完整URL>)`
- 注释宽350、高70

**说明区域（数据行下方）：**
```
说明:
1. 每个单元格右上角红三角 = 来源注释，同一行不同列来源不同。
2. 火灾危险性类别仅供参考，实际分类应以当地消防部门核定为准。
3. 仅标注甲/乙/丙类，非可燃物质统一标注为"非可燃"。
4. 职业接触限值依据 GBZ 2.1-2019，单位 mg/m³。
5. 危化品类别依据《危险化学品目录》(2015版)。
6. 因牌号/结构未定的物质，标注"因牌号而异"，需查阅实际供应商SDS。
7. 有毒气体检测目录依据《高毒物品目录》、GB/T 50493-2019附录B、HG/T 20660-2017附录A。
```

## 5. 文件输出

```

```

**保存位置：** 用户桌面（`C:\Users\<username>\Desktop\`）或用户指定路径。

**编码问题处理（Windows）：**
- 中文文件名在 openpyxl 写入时可能损坏
- **解决方案：先保存为纯英文临时文件名，再 os.rename 为中文名**
- 保存后立即用 zipfile.ZipFile 校验文件完整性

```python
# 安全写入模式
tmp = os.path.join(output_dir, "chem_tmp.xlsx")
wb.save(tmp)
import zipfile
zipfile.ZipFile(tmp).close()  # 校验
os.rename(tmp, final_path)
```

**脚本清理：** 生成完毕后删除临时 Python 脚本。

## 6. 异常处理

| 场景 | 处理 |
|------|------|
| 某化学品CAS查不到 | 标注"数据未公开"，不中断 |
| 某属性无数据 | 标注"—"或"数据未公开" |
| 导热油等通用名 | 标注"因牌号而异"，提示查供应商SDS |
| 用户仅需部分字段 | 按用户要求裁剪列 |
| 文件被Excel锁定 | 换文件名（加_v2/_v3后缀） |
| 数据来源不确定 | 标注"计算值"或"供应商SDS" |
| 工艺专有中间体 | 物理常数"数据未公开"，火灾类别"丙类(估)"，来源注明"工艺专有，按有机物估计" |
| Agent查询中断（JSON Parse error） | Agent将结果写入文件而非返回文本；用SendMessage继续中断的Agent |
| 化学品名称变体/错别字 | 建立NORM映射表统一归并，保留原写法作为"变体"列 |
| 同一物质出现在多个设备/工段 | 合并去重，统计"出现设备数(去重)"和"涉及工段" |
