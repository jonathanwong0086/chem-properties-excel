# 输出 Excel 结构说明

生成的 Excel 文件包含两个 Sheet。

## Sheet 1：物性数据主表

### 表头（16列）

| 列号 | 列名 | 数据类型 | 颜色 |
|------|------|----------|------|
| A | 序号 | 整数 | 无 |
| B | 化学品名称 | 文本 | **着色**（甲红/乙橙/丙黄/白） |
| C | CAS号 | 文本 | **着色**（同上） |
| D | 熔点(°C) | 文本 | 无 |
| E | 沸点(°C) | 文本 | 无 |
| F | 闪点(°C) | 文本 | 无 |
| G | 密度 | 文本 | 无 |
| H | 常温物态 | 固/液/气 | 无 |
| I | 火灾危险性类别 | 甲类/乙类/丙类/非可燃 | **着色**（同上） |
| J | 水溶性 | 描述文本 | 无 |
| K | 密度>水？ | 是/否 | 无 |
| L | 禁用水灭火？ | 是/否（附原因） | 无 |
| M | 爆炸极限(LEL~UEL) | 百分比范围 | 无 |
| N | 蒸汽>空气？ | 是/否(MR值) | 无 |
| O | 职业接触限值(mg/m³) | TWA/STEL/MAC 值 | 无 |
| P | 危化品类别 | 类别描述 | 无 |

### 着色规则

仅 **B列（名称）、C列（CAS号）、I列（火灾危险性类别）** 三列着色：

| 火灾类别 | 颜色 | 判定依据 |
|----------|------|----------|
| 甲类 | 红底 (#FFCCCC) | 闪点<28°C液体 / 爆炸下限<10%气体 |
| 乙类 | 橙底 (#FFE0B2) | 28°C≤闪点<60°C液体 |
| 丙类 | 黄底 (#FFF9C4) | 闪点≥60°C液体 / 可燃固体 |
| 非可燃 | 白底 (#FFFFFF) | 不可燃物质 |

### 来源注释

每个数据单元格（除序号和名称外）右上角有红色三角标记，鼠标悬停可查看：

```
数据来源:
PubChem (https://pubchem.ncbi.nlm.nih.gov/compound/75-05-8)
```

**同一行不同列的注释来源不同**，例如乙腈行：
- CAS号 → PubChem
- 熔点 → NIST Chemistry WebBook
- 闪点 → Sigma-Aldrich SDS
- 爆炸极限 → NIOSH Pocket Guide
- OEL → GBZ 2.1-2019

### 说明区域

数据行下方包含图例和说明文字。

## Sheet 2：数据来源汇总

### 表头（4列）

| 列名 | 内容 |
|------|------|
| 类型 | 国际数据库/化学品供应商/中国国家标准等 |
| 机构/网站名称 | 完整机构名 |
| 本次提供数据字段 | 该来源提供了哪些字段 |
| 网址 | 可点击的超链接 |

### 内容

固定12个来源，涵盖物化属性查询、安全数据、中国法规三大类。

## 格式规格

| 属性 | 值 |
|------|------|
| 字体 | 微软雅黑 |
| 表头字号 | 10pt 加粗 白字 |
| 数据字号 | 9pt |
| 行高 | 统一 30px（表头 32px） |
| 列宽 | 5~26 字符（紧凑不遮盖） |
| 冻结窗格 | 首行 |
| 对齐 | 数值居中，文本左对齐，自动换行 |
